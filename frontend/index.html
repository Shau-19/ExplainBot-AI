How It Works

1. RETRIEVAL

BM25 retrieves only the most relevant document chunks — never hallucination-prone full-context

2. DECISION

Agent analyzes query and chooses format — text, audio, or video

3. GENERATION

ElevenLabs or OpenAI TTS creates per-scene audio in the detected language

4. COMPOSITION

MoviePy composes scenes using real audio durations for perfect A/V sync

KEY INNOVATION

Grounded Retrieval: Answers come from the document, not the LLM's imagination. BM25 finds the right chunks. If the answer isn't there, it says so.
